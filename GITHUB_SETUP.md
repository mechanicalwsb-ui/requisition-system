# คู่มือการติดตั้งระบบบน GitHub Pages (ภาษาไทย) 🚀

ระบบเบิกพัสดุนี้ได้รับการปรับปรุงโครงสร้างให้แยกส่วน **หน้าบ้าน (Frontend)** ไปทำงานบน **GitHub Pages** เพื่อให้สมาร์ทโฟนทุกระบบปฏิบัติการ (iOS/Android) และทุกเบราว์เซอร์สามารถเรียกใช้งานกล้องสแกนได้แบบ 100% โดยไม่มีข้อจำกัดความปลอดภัยของ Google iFrame อีกต่อไป!

ทำตามขั้นตอนด้านล่างนี้เพื่อนำระบบขึ้นใช้งานจริง:

---

## ขั้นตอนที่ 1: เตรียม Google Apps Script ให้พร้อม

1. เปิดหน้า **Google Sheets** ของระบบเบิกพัสดุ
2. ไปที่ **ส่วนขยาย (Extensions)** -> **Apps Script**
3. คัดลอกโค้ดจากไฟล์ `Code.gs` ทั้งหมดในโฟลเดอร์นี้ ไปวางทับโค้ดเดิมในเว็บ Apps Script
4. กดบันทึกโครงการ (รูปแผ่นดิสก์)
5. **ทำการ Deploy ใหม่** (สำคัญมาก):
   - กดปุ่ม **การทำให้ใช้งานได้ (Deploy)** ที่มุมขวาบน -> เลือก **จัดการการทำให้ใช้งานได้ (Manage deployments)**
   - กดรูปดินสอ ✏️ หลัง Deployment เดิม
   - ในช่อง **เวอร์ชัน (Version)** ให้เลือก **เวอร์ชันใหม่ (New version)**
   - ตรวจสอบให้มั่นใจว่าช่องผู้มีสิทธิ์เข้าถึงคือ **ทุกคน (Anyone)**
   - กด **ทำให้ใช้งานได้ (Deploy)**
   - **คัดลอก URL ของเว็บแอป (Web App URL)** ที่ได้มา (ลิงก์ที่ลงท้ายด้วย `/exec`)

---

## ขั้นตอนที่ 2: ตั้งค่า URL และรหัสผ่านที่หน้าบ้าน

1. เปิดไฟล์ `gh-pages/config.js` ในคอมพิวเตอร์ของคุณ
2. ตรงช่อง `GAS_URL` ให้วาง URL ของ Apps Script ที่คุณคัดลอกมาจากขั้นตอนก่อนหน้า
3. ตรงช่อง `PASSWORD` ให้ใส่รหัสผ่านที่คุณต้องการตั้งเพื่อควบคุมสิทธิ์ก่อนเข้าสู่หน้าระบบสแกน (รหัสเริ่มต้นคือ `"1234"`)
4. บันทึกไฟล์ `config.js`

---

## ขั้นตอนที่ 3: นำไฟล์ทั้งหมดในโฟลเดอร์ `gh-pages` ขึ้น GitHub

หากคุณยังไม่มีโปรเจกต์ GitHub Pages ให้ทำตามขั้นตอนนี้:

### วิธีอัปโหลดผ่านเว็บเบราว์เซอร์ (ง่ายที่สุด ไม่ต้องลงโปรแกรม)
1. เข้าเว็บไซต์ [GitHub.com](https://github.com/) และล็อกอินเข้าสู่ระบบ (หากยังไม่มีให้สมัครสมาชิกฟรี)
2. สร้าง Repository ใหม่:
   - กดปุ่ม **New** (หรือเครื่องหมายบวก `+` ที่มุมขวาบน เลือก **New repository**)
   - ตั้งชื่อโปรเจกต์ในช่อง **Repository name** (เช่น `requisition-system`)
   - เลือกสถานะเป็น **Public** (จำเป็นต้องเลือก Public สำหรับใช้งาน GitHub Pages รุ่นฟรี)
   - ติ๊กถูกที่ช่อง **Add a README file**
   - กดปุ่ม **Create repository**
3. อัปโหลดไฟล์:
   - กดปุ่ม **Add file** ที่มุมขวาบนของหน้าต่างโปรเจกต์ -> เลือก **Upload files**
   - ลากไฟล์ทั้งหมดที่อยู่ **ภายในโฟลเดอร์ `gh-pages`** มาวางในหน้านี้ (ประกอบไปด้วย: `index.html`, `style.css`, `config.js`, `app.js`)
   - รอจนไฟล์ทั้งหมดอัปโหลดเสร็จ
   - กดปุ่ม **Commit changes** ที่ด้านล่าง

---

## ขั้นตอนที่ 4: เปิดใช้งาน GitHub Pages

1. ในหน้าโปรเจกต์ GitHub ของคุณ ไปที่เมนู **Settings** (รูปฟันเฟืองด้านบน)
2. เลื่อนเมนูด้านซ้ายลงมาหาหัวข้อ **Code and automation** -> คลิกเลือก **Pages**
3. ในหัวข้อ **Build and deployment**:
   - ตรงส่วน **Source** ให้เลือกเป็น **Deploy from a branch**
   - ตรงส่วน **Branch** ให้เปลี่ยนจาก `None` เป็น **`main`** (หรือ `master`) และโฟลเดอร์ย่อยเป็น **`/ (root)`**
   - กดปุ่ม **Save**
4. รอประมาณ 1-2 นาที ให้รีเฟรชหน้าจอนี้ ระบบจะแสดงลิงก์เว็บไซต์ของคุณที่ด้านบนสุด เช่น:
   `https://<ชื่อผู้ใช้ของคุณ>.github.io/<ชื่อโปรเจกต์ของคุณ>/`

---

## 🎉 การใช้งานบนโทรศัพท์มือถือ
- นำลิงก์จาก GitHub Pages ที่ได้ ไปเปิดในสมาร์ทโฟนเครื่องใดก็ได้
- เมื่อเปิดขึ้นมา ระบบจะถามรหัสผ่าน (ที่ตั้งไว้ใน `config.js`)
- เมื่อกรอกรหัสผ่านผ่านแล้ว ระบบจะขอสิทธิ์เข้าถึงกล้อง ให้กด **อนุญาต (Allow)**
- จากนั้นสามารถกด **"เปิดใช้งานกล้องวิดีโอสด"** หรือ **"ถ่ายรูปเพื่อสแกน"** เพื่อทำรายการได้ทันที!
- ระบบทำงานได้เสถียร 100% ทั้งบน Safari (iOS), Chrome (Android) และ Samsung Internet
